# 快速部署指南 - 5分钟搞定你的专属工作时长统计网站

本指南将帮助你快速将工作时长统计网站部署到云端，实现数据云端存储、多设备同步。

---

## 方案一：Railway 部署（推荐，最简单）

Railway 提供免费额度，完全足够个人使用。

### 步骤 1：准备代码

1. 将本项目代码上传到你的 GitHub 仓库
   - 如果没有 GitHub 账号，先注册一个：https://github.com
   - 创建一个新仓库，把项目代码上传进去

### 步骤 2：注册 Railway 账号

1. 访问 https://railway.app
2. 点击 "Start a New Project"
3. 选择 "Login with GitHub"，用你的 GitHub 账号登录

### 步骤 3：部署项目

1. 登录后，点击 "New Project"
2. 选择 "Deploy from GitHub repo"
3. 选择你刚才创建的仓库
4. 点击 "Deploy Now"
5. 等待部署完成（大约1-2分钟）

### 步骤 4：配置环境变量

1. 部署完成后，进入项目的 "Variables" 页面
2. 添加一个新变量：
   - 变量名：`JWT_SECRET`
   - 变量值：随便输入一串随机字符（比如 `my-secret-key-123456`，越复杂越安全）
3. 点击 "Add" 保存

### 步骤 5：获取访问地址

1. 进入项目的 "Settings" 页面
2. 找到 "Domain" 部分
3. 点击 "Generate Domain" 生成一个免费域名
4. 或者绑定你自己的域名

🎉 **完成！** 现在你可以通过这个域名访问你的工作时长统计网站了。

## 方案二：Render 部署（免费）

Render 也是一个不错的选择，免费套餐足够个人使用。

### 步骤 1：准备代码

同方案一，将代码上传到 GitHub 仓库。

### 步骤 2：注册 Render 账号

1. 访问 https://render.com
2. 点击 "Get Started"
3. 用 GitHub 账号登录

### 步骤 3：创建 Web Service

1. 点击 "New" → "Web Service"
2. 选择你的 GitHub 仓库
3. 填写配置：
   - **Name**: 随便起个名字（比如 `work-hours-tracker`）
   - **Runtime**: `Node`
   - **Build Command**: `cd server && npm install`
   - **Start Command**: `cd server && node server.js`
4. 选择免费套餐（Free）
5. 点击 "Advanced" → "Add Environment Variable"
   - Key: `JWT_SECRET`
   - Value: 你的自定义密钥
6. 点击 "Create Web Service"

### 步骤 4：等待部署

等待 2-3 分钟，部署完成后会自动分配一个 `.onrender.com` 域名。

🎉 **完成！** 通过这个域名即可访问。

## 方案三：本地运行（适合测试）

如果你只是想在自己电脑上运行，可以按以下步骤操作：

### 前置要求
- 安装 Node.js（版本 18 或更高）
  - 下载地址：https://nodejs.org

### 步骤

1. 打开命令行（CMD 或终端）
2. 进入项目的 `server` 目录：
   ```bash
   cd server
   ```
3. 安装依赖：
   ```bash
   npm install
   ```
4. 启动服务：
   ```bash
   node server.js
   ```
5. 打开浏览器访问：http://localhost:3000

## 常见问题

### Q: 部署后数据会丢失吗？
A: 数据存储在服务器的 SQLite 数据库文件中。只要不删除部署实例，数据就不会丢失。建议定期备份数据库文件。

### Q: 免费额度够用吗？
A: 完全够用！Railway 和 Render 的免费套餐对于个人使用来说绰绰有余。

### Q: 可以多人使用吗？
A: 可以！每个用户注册自己的账号，数据独立存储，互不影响。

### Q: 数据安全吗？
A: 密码使用 bcrypt 加密存储，即使数据库泄露也无法破解密码。建议设置复杂的 JWT_SECRET。

### Q: 可以修改界面风格吗？
A: 可以！前端代码在 `public` 目录下，你可以自由修改 HTML、CSS、JS 文件。
