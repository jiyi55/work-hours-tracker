# 工作时长统计 - 使用说明

## 一、功能介绍

本网站是一款专为铁路乘务人员设计的工作时长统计工具，**数据存储在云端服务器**，支持多设备登录同步。

### 核心功能

1. **用户系统**：邮箱注册登录，数据独立存储
2. **云端同步**：无论在哪台设备登录，数据都能同步显示
3. **月度统计**：
   - 月总工时（目标100小时）
   - 关键时长（目标50小时）
   - 出勤趟数统计
4. **关键时段计算**：
   - 午间时段：12:00 - 14:00
   - 夜间时段：22:00 - 次日06:00
5. **小添乘功能**：一键增加16小时工时
6. **出勤记录管理**：
   - 手动添加出勤/退勤时间
   - 自动计算总工时和关键时长
   - 支持删除记录
7. **月份切换**：可查看历史月份数据

## 二、快速开始

### 1. 注册账号

1. 打开网站，点击"注册"标签
2. 输入你的邮箱地址
3. 设置密码（至少6位）
4. 再次输入密码确认
5. 点击"注册"按钮

注册成功后会自动登录。

### 2. 登录账号

1. 打开网站，在"登录"标签下
2. 输入注册时的邮箱和密码
3. 点击"登录"按钮

登录成功后即可进入主页面。

### 3. 退出登录

点击右上角的"退出"按钮，即可返回登录页面。

---

## 三、使用指南

### 1. 添加出勤记录

1. 在"添加出勤记录"区域输入时间
2. **出勤时间**：格式为4位数字，如 `0800` 表示 08:00
3. **退勤时间**：格式为4位数字，如 `2000` 表示 20:00
4. 点击"+ 添加"按钮
5. 系统自动计算总工时和关键时长，并保存到云端

**注意**：
- 支持跨天班次，如出勤 2200，退勤 0600，系统会自动计算为8小时
- 记录会自动计入当前显示的月份

### 2. 查看统计数据

页面上方三个卡片实时显示：
- **月总工时**：本月所有出勤记录的总时长
- **关键时长**：落在关键时段内的工作时长
- **出勤趟数**：本月出勤次数

每个卡片下方显示距离目标值还差多少。

### 3. 小添乘功能

勾选"小添乘 (+16小时)"复选框，总工时会自动增加16小时。
适用于小添乘任务的工时统计。

### 4. 切换月份

- 点击月份左侧的 `‹` 按钮：查看上一个月
- 点击月份右侧的 `›` 按钮：查看下一个月

**注意**：添加记录时，会自动计入当前显示的月份。如需补录历史记录，请先切换到对应月份再添加。

### 5. 删除记录

在出勤记录列表中，点击每条记录右侧的 🗑️ 图标，确认后即可删除该条记录。
删除后数据会从云端同步更新。

## 四、关键时段说明

### 什么是关键时段？

关键时段是指需要重点统计的特殊工作时间段，包含两个时段：

1. **午间时段**：12:00 - 14:00（2小时）
2. **夜间时段**：22:00 - 次日06:00（8小时）

### 计算规则

系统会自动计算你的工作时间与关键时段的重叠部分：

- **例1**：出勤 08:00，退勤 20:00
  - 总工时：12小时
  - 关键时长：2小时（12:00-14:00）

- **例2**：出勤 22:00，退勤 06:00（次日）
  - 总工时：8小时
  - 关键时长：8小时（全部在关键时段内）

- **例3**：出勤 18:00，退勤 02:00（次日）
  - 总工时：8小时
  - 关键时长：4小时（22:00-02:00）

## 五、云端数据同步说明

### 数据存储位置

所有数据都存储在服务器的数据库中，而不是你的浏览器本地。

### 同步特点

✅ **多设备同步**：在电脑上添加的记录，手机上登录同一账号就能看到
✅ **数据安全**：密码加密存储，不用担心泄露
✅ **不会丢失**：清除浏览器缓存、换设备都不会丢失数据
✅ **随时访问**：只要有网络，随时随地都能查看和管理

### 注意事项

1. 请妥善保管你的账号密码
2. 建议使用常用邮箱注册，方便记忆
3. 不要在公共设备上勾选"记住密码"（本系统暂不支持记住密码功能）

## 六、常见问题

**Q: 忘记密码怎么办？**
A: 目前暂不支持密码找回功能。建议使用常用邮箱注册，并牢记密码。如确实忘记，只能重新注册新账号。

**Q: 时间格式怎么输入？**
A: 输入4位数字，不需要冒号。如上午9点30分输入 `0930`，下午6点输入 `1800`。

**Q: 跨天的班次怎么计算？**
A: 系统会自动识别跨天情况。如晚上10点上班，第二天早上6点下班，输入 `2200` 和 `0600` 即可。

**Q: 可以修改已添加的记录吗？**
A: 目前暂不支持修改，如需调整请先删除错误记录，再重新添加正确的。

**Q: 小添乘是什么意思？**
A: 小添乘是铁路系统的一种工作形式，通常一次性计16小时工时。勾选后会在总工时中增加16小时。

**Q: 数据会泄露吗？**
A: 数据存储在服务器数据库中，密码采用加密存储。只要你保管好自己的账号密码，数据是安全的。

**Q: 换了手机/电脑，数据还在吗？**
A: 在的！只要登录同一个账号，所有数据都会同步显示，不受设备更换影响。

## 七、部署方式（技术向）

如果你想自己部署这个网站，可以选择以下方式：

### 方式一：Railway 部署（推荐）

1. 访问 https://railway.app 注册
2. 新建项目，上传代码
3. 设置环境变量 `JWT_SECRET`
4. 等待自动部署完成

### 方式二：Render 部署

1. 访问 https://render.com 注册
2. 创建 Web Service
3. 配置构建和启动命令
4. 添加环境变量

### 方式三：本地运行

```bash
cd server
npm install
node server.js
```

详细部署说明请查看 `README.md` 文件。
